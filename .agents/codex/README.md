# Codex Instruction Mirror

This directory stores AI-oriented instruction mirrors for this repository.

- Human-first source docs remain in `AGENTS.md` and `context/`.
- Machine-friendly mirrors live in `.agents/codex/instructions/`.
- Sync mode is manual; update mirrors when source docs change.

## Why `.agents/codex` instead of `.codex`

In this repository environment, `.codex` is an existing busy file and cannot be replaced with a directory. This path is the compatible fallback.
