# Backend Role Playbook

## Target positioning

Backend engineer with production microservices experience, measurable optimization outcomes, and strong systems-level fundamentals.

## Prioritize these signals

- E-003 microservices architecture and scale.
- E-001 workflow redesign with measurable speedup.
- E-004 delivery automation supporting backend reliability.
- E-008 interpreter/compiler project rigor.

## Secondary signals

- E-005 autoscaling operations.
- E-007 systems-depth credibility.

## De-prioritize

- Frontend-centric copy.
- Broad tool dumps with no architecture context.

## Recommended summary shape

- Line 1: backend architecture and service ownership.
- Line 2: performance/throughput improvement evidence.
- Line 3: reliability and maintainability orientation.

## Skills emphasis order

1) Node.js, microservices, API systems
2) TypeScript, Go, Python, Rust
3) PostgreSQL, MongoDB
4) CI/CD and container infrastructure
