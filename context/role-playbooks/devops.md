# DevOps Role Playbook

## Target positioning

Platform and DevOps-oriented engineer with strong ownership of CI/CD, cloud infrastructure, and scalable deployment patterns.

## Prioritize these signals

- E-004 CI/CD pipeline ownership.
- E-005 autoscaling and production service reliability.
- E-003 scale growth to 100K users.
- E-001 for throughput improvement in production workflow.

## Secondary signals

- E-002 for operational guardrails.
- E-006 for automation mindset.

## De-prioritize

- Low-level project depth unless role values systems.
- AI buzzwords without operational context.

## Recommended summary shape

- Line 1: platform reliability + cloud execution.
- Line 2: CI/CD and deployment automation ownership.
- Line 3: scaling outcomes and operational discipline.

## Skills emphasis order

1) GCP, Docker, Kubernetes
2) GitHub Actions, Cloud Build, CI/CD
3) Node.js, microservices
4) PostgreSQL, MongoDB
